## 1.Description

[Nest](https://github.com/nestjs/nest) Application used to keep track of quotes for industrial instrumentation equipment.

## 2.Clone project

here should be the url of repository

## 3.Requirements
  - install Nest CLI
  ```npm i -g @nestjs/cli```
  - build the database from dockercompose
  ```docker compose up -d```


## 4.Installation

```bash
$ yarn install or npm run install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
